# JinjaRender
A minimalist API to generate HTML from a Jinja template and a JSON file


You have 2 options:

## Python
```
git clone https://github.com/hoolies/JinjaRender/
python -m venv .
pip install -r requirements.txt
bin/python app.py
```
## Docker
```
git clone https://github.com/hoolies/JinjaRender/
bash ./build/deploy.sh
```
